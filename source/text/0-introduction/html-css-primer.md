# HTML and CSS Primer

[&laquo; Return to the Chapter Index](index.md)

## Why this appears early

We use browser examples throughout the course, so you need lightweight HTML and CSS basics before Chapter 9.

## What you need to know now

- HTML elements define page structure.
- CSS rules style those elements.
- JavaScript/TypeScript can query and update them through the DOM.

At this stage, focus on:

- Structural tags (`div`, `section`, `main`, headings, paragraphs)
- Input tags (`button`, `input`, `select`)
- CSS selectors (class and id)
- Basic layout (`display`, `margin`, `padding`)

## What can wait

You do not need full front-end engineering details yet. We will add complexity gradually as OOP topics advance.

## Summary

A small amount of HTML/CSS knowledge is enough to begin connecting CS2 topics to browser APIs.
