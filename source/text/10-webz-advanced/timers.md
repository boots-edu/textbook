## Key Idea

Timed updates let programs animate visuals, run game loops, and schedule audio/interaction events.

## Timing APIs

- `setTimeout` for one-time delayed actions
- `setInterval` for repeated actions
- `requestAnimationFrame` for smooth visual updates

## Canvas Essentials

`<canvas>` gives a drawing surface for:

- simple arcade interactions
- shape-based animation
- sprite-like rendering with images

## Example Activity: Click Arcade Prototype

Students implement:

- A moving target drawn in canvas
- Score and countdown timer
- Start/stop/reset controls

## Audio Extension

Use Web Audio API for feedback sounds:

- success tone on hit
- miss tone on incorrect clicks

## Summary

Timing, canvas, and audio APIs support engaging projects while still reinforcing classes, state updates, and event-driven design.
