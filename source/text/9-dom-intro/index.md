# Chapter 9 - The Browser DOM

[&laquo; Return to the Table of Contents](../../index.md)

The browser gives us one of the richest object-oriented systems most programmers will ever use: the **Document Object Model**, or **DOM**. A web page is not just raw text after the browser loads it. It becomes a hierarchy of objects with fields, methods, inheritance relationships, and events.

In this chapter, we will learn how HTML and CSS appear inside the browser, how the `document` object represents a page, and how TypeScript can query and update that object structure. Along the way, we will connect these browser APIs back to the same course ideas we have already studied: classes, inheritance, abstract classes, polymorphism, trees, and higher-order functions.
