# Chapter 2 - Loops and Arrays

[&laquo; Return to the Table of Contents](../../index.md)

Loops and iteration are key concepts in programming. Time to learn how loops work and how to loop through arrays and objects in TypeScript!

## Browser API Framing

Browser interfaces are full of collections. Menus, lists, tables, cards, and form controls are often handled as groups, and loops are how we process those groups efficiently.

In browser work, you will frequently iterate over:

- Arrays of application data (for example, scores, tasks, or rows from a file)
- Element collections returned from the DOM (for example, a list of matching nodes)
- Parallel arrays that link model data to rendered output

Rendering is usually just structured iteration: for each data item, create or update the matching UI element. This chapter's loop patterns directly become your rendering patterns later in the book.

When you practice loops here, keep asking: "What if each element in this array were a visible row in a web page?" That mindset makes the transition to dynamic interfaces much smoother.
