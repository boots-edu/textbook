# Chapter 1 - Introduction To TypeScript

[&laquo; Return to the Table of Contents](../../index.md)

In this chapter, we are going to explain the basics of TypeScript.
This will include variables, values, types, and functions.

## Browser API Framing

These same fundamentals are the building blocks for browser programming.
When we interact with the DOM, we still rely on variables, values, and functions:

- Variables store references to elements we find on the page.
- Types help us distinguish generic elements from specific ones like buttons and inputs.
- Functions package behavior we run when events occur.

A browser program can be read as: "get a value from the page, compute a new value, and write the result back to the page." That cycle depends directly on the core language skills in this chapter.

As you work through the chapter, think of each example twice: first as plain TypeScript logic, and second as logic that could run inside a web page to read user input, validate data, and update visible output.
