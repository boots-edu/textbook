# Chapter 10 - Advanced Browser APIs

[&laquo; Return to the Table of Contents](../../index.md)

The browser contains far more than a few buttons and text boxes. In this chapter we will continue treating the browser as an object-oriented environment and look at some of its more interesting native APIs. We will define our own custom elements, build dynamic interfaces with plain DOM operations, pass messages with custom events, draw graphics with the Canvas API, and create sound with the Web Audio API.
